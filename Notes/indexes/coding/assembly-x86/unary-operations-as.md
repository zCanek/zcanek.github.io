These are unary operations, with the single [[Operands - as|operand]] serving as both source and destination. This operand can be either a [[register]] or a memory location.

For example, the instruction `incq (%rsp)` causes the 8-byte element element on the top of the [[program stack|stack]] to be incremented. This syntax is reminiscent of the C increment (++) and decrement (--) operators.
