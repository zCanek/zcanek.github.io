Boolean Algebra

Boolean algebra deals with Boolean (also called binary) values that are typically labeled true/false, 1 or 0.

Boolean Function

A Boolean function is a function that operates on binary inputs and returns binary outputs.

The simplest way to specify a Boolean function is to enumerate all the possible values of the function’s input variables, along with the function’s output for each set of inputs. This is called the truth table representation of the function...

Boolean Expression

Boolean function can also be specified using Boolean operations over its input variables.
The basic Boolean operator that are typically used are:

  • XOR
  • OR
  • NOT
  • AND
  • NAND
  • NOR
